# Deploying BERT on Heroku

Code for the medium blog post [Deploying BERT on Heroku](https://pages.github.com/).

## Local Deployment
### Requirements

Install the required libraries using

`pip install -r requirements.txt`

### Inference
Run the server using

`python app.py`

Then go this url
http://localhost:5000
